# How to install

1. `composer install`

# Requirements

1. Imagick
   1. `sudo apt-get install php-imagick`
2. PHP-zip
   1. `sudo apt-get install php-zip`
3. ffmpeg
   1. `sudo apt install software-properties-common`
   2. `sudo apt update`
   3. `sudo add-apt-repository ppa:jonathonf/ffmpeg-4`
   4. `sudo apt install ffmpeg`
   5. `ffmpeg -version`
